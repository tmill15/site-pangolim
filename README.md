# Pangolim Chinês — Projeto Escolar

Site educacional sobre o **Pangolim Chinês** (*Manis pentadactyla*), desenvolvido como trabalho escolar do CIMAN.

## Sobre

O site apresenta informações sobre o pangolim chinês: aparência, habitat, alimentação, comportamento defensivo, filhotes, ameaças de extinção e curiosidades. Inclui também um quiz interativo e uma galeria de fotos.

## Tecnologias

- HTML, CSS e JavaScript (vanilla, arquivo único)
- Google Fonts (Baloo 2 + DM Sans)
- Deploy via [Vercel](https://vercel.com)
