# Pangolim Chinês — Projeto Escolar

Site educacional sobre o **Pangolim Chinês** (*Manis pentadactyla*), desenvolvido como trabalho escolar do CIMAN.

## Sobre

O site apresenta informacoes sobre o pangolim chines: aparencia, habitat, alimentacao, comportamento defensivo, filhotes, ameacas de extincao e curiosidades. Inclui tambem um quiz interativo e uma galeria de fotos.

## Tecnologias

- HTML, CSS e JavaScript (vanilla, arquivo unico)
- Google Fonts (Baloo 2 + DM Sans)
- Deploy via [Vercel](https://vercel.com)
